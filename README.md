## ✨ Features

### 🐍 Python Section
- **55+ coding questions** across 6 core topics
- Every question has two tabs:
  - 💻 **Solution** — Full working Python code
  - 💡 **Concept** — Plain-English explanation of the logic and pattern behind it
- Filter by **topic** or **difficulty** (Easy / Medium / Hard)
- **Search bar** to find any question instantly
- **Mark as Done** — tracks your progress with a live progress bar
- Progress is **saved in your browser** — picks up where you left off

### 🧮 Aptitude Section
- All **18 concepts** tested in every tech interview
- Each concept includes:
  - 📐 Formula / Key concept summary
  - 4 MCQ practice questions with instant feedback
  - Full explanation for every answer
